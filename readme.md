# 项目介绍

这是一个企业管理项目

# 所需环境

前端：Node.js@16

后端: Java@8、Mysql@5.8、SpringBoot@2.2.4

运维: ngnix

# 安装教程

## 1. 导入Sql语句

新建数据库guoran

把sql导入到guoran数据库库

## 2. 启动后端

Jar包启动

```sh
java -jar guoran1.0.jar
```

项目启动
....

## 3. 启动前端

0. 克隆本仓库
1. 进入启动项目

```cmd
cd guoran/web
```

2. 安装依赖

```cmd
npm i
```

3. 运行

```cmd
npm run dev
```
